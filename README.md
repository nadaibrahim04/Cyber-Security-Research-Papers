## Research Papers

## 1. Autoencoder-Based Phishing Website Detection Using URL and Content Features

**Topic:** Deep Learning & Cybersecurity  
**Authors:** Nada Ibrahim, Lana Alnajjar  
**Supervisor:** Mehak Khurana  
**Institution:** Canadian University Dubai  

This research explores a deep learning approach for detecting phishing websites using URL and content-based features. The study investigates the use of an autoencoder model for identifying malicious websites and evaluates its effectiveness compared to other machine learning approaches.

**Key Areas:**
- Phishing website detection
- Deep learning in cybersecurity
- Autoencoder-based anomaly detection
- Feature extraction
- Machine learning evaluation
